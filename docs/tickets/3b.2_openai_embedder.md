# Ticket 3b.2: Add OpenAI embedder implementation

## Summary
Implement OpenAI text embedding provider using `text-embedding-3-small` model.

## Context
OpenAI is the most popular embedding provider. Easy to integrate via their API.

## Acceptance Criteria
- [ ] `OpenAIEmbedder` implements `domain.Embedder`
- [ ] Uses `text-embedding-3-small` model (1536 dims, configurable)
- [ ] Auth via OPENAI_API_KEY environment variable
- [ ] Retry with exponential backoff on errors
- [ ] Unit tests with mocked HTTP client

## Technical Details
- API: `POST https://api.openai.com/v1/embeddings`
- Auth: `OPENAI_API_KEY` env var
- Model: `text-embedding-3-small` (1536 dims, can reduce to 384)
- Timeout: 30 seconds
