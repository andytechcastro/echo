# Ticket 3b.3: Add Cohere embedder implementation

## Summary
Implement Cohere text embedding provider using `embed-v3` model.

## Context
Cohere offers high-quality embeddings with good multilingual support.

## Acceptance Criteria
- [ ] `CohereEmbedder` implements `domain.Embedder`
- [ ] Uses `embed-v3` model (1024 dims)
- [ ] Auth via COHERE_API_KEY environment variable
- [ ] Retry with exponential backoff on errors
- [ ] Unit tests with mocked HTTP client

## Technical Details
- API: `POST https://api.cohere.ai/v1/embed`
- Auth: `COHERE_API_KEY` env var
- Model: `embed-v3` (1024 dims)
- Input type: `search_document` (for save) or `search_query` (for search)
- Timeout: 30 seconds
