# Ticket 3b.1: Add Vertex AI embedder implementation

## Summary
Implement Vertex AI text embedding provider using `text-embedding-004` model.

## Context
Phase 3b adds external API support. Vertex AI is the primary provider for GCP users.

## Acceptance Criteria
- [ ] `VertexAIEmbedder` implements `domain.Embedder`
- [ ] Uses `text-embedding-004` model (768 dims)
- [ ] Auth via Application Default Credentials
- [ ] Retry with exponential backoff on errors
- [ ] Unit tests with mocked HTTP client

## Technical Details
- API: `POST https://{region}-aiplatform.googleapis.com/v1/projects/{project}/locations/{region}/publishers/google/models/text-embedding-004:predict`
- Auth: Google ADC (service account or user credentials)
- Model: `text-embedding-004` (768 dimensions)
- Timeout: 30 seconds
